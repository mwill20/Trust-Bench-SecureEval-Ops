Trust_Bench Tag Extraction Demo (single-file)
--------------------------------------------
Files:
- trustbench_tag_extractor.py : main demo script
- data/gazetteer.csv : small curated gazetteer
- fixtures/ : sample project blurbs

Run demo:
python trustbench_tag_extractor.py --demo
Or analyze a file:
python trustbench_tag_extractor.py --file fixtures/vuln_project.txt
